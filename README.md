# CSS Projekty a Komponenty 🎨

Tento repozitář obsahuje sbírku mých různých CSS projektů, komponent a experimentů. Každá složka představuje samostatný projekt nebo ukázku určité techniky či designového prvku.

## 📂 Obsah repozitáře

Zde je přehled všech projektů, které v tomto repozitáři najdete:

*   **[animations](./animations/)**: Ukázky různých CSS animací.
*   **[buttons](./buttons/)**: Různé styly a efekty pro tlačítka.
*   **[card roundclicker](./card%20roundclicker/)**: Interaktivní karta.
*   **[error 404 page](./error%20404%20page/)**: Design stránky pro chybu 404.
*   **[feedback page](./feedback%20page/)**: Stránka pro uživatelskou zpětnou vazbu.
*   **[monsters](./monsters/)**: CSS kresby nebo animace (monstra).
*   **[pricing cards](./pricing%20cards/)**: Ceníkové karty.
*   **[register and login](./register%20and%20login/)**: Formuláře pro registraci a přihlášení.
*   **[resposvive sviper](./resposvive%20sviper/)**: Responzivní slider/swiper komponenta.
*   **[search bar](./search%20bar/)**: Různé designy vyhledávacích polí.
*   **[social media icons](./social%20media%20icons/)**: Ikonky sociálních sítí a efekty na ně.

## 🚀 Jak projekty spustit

Většina těchto projektů se skládá pouze z HTML, CSS a občas trochu JavaScriptu. Pro zobrazení stačí:
1. Naklonovat nebo stáhnout tento repozitář.
2. Otevřít složku konkrétního projektu.
3. Dvojklikem otevřít soubor `index.html` ve vašem oblíbeném webovém prohlížeči.

## 🛠️ Technologie

*   HTML5
*   CSS3 (Včetně Flexboxu, Gridu, Animací a Transitions)
*   Vanilla JavaScript (u některých interaktivních komponent)
